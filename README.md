## 🌍 Overview

**LexAI** is a client-side legal assistant web app powered by Google Gemini. It helps users understand their legal rights across **40+ countries** in three languages: English, French, and Arabic.

It is **not** a substitute for licensed legal counsel — it provides structured legal information to help users ask better questions and make informed decisions.

---

## ✨ Features

- 🔒 **Multi-domain coverage** — Criminal, Civil, Family, Labor, Business, Real Estate, Immigration, Tax, Constitutional, and Islamic/Arab law
- 🌍 **Multi-jurisdiction** — USA, UK, France, Morocco, Germany, Canada, Australia, UAE, and more
- 🗣️ **Trilingual** — Detects and responds in English, French, or Arabic automatically
- 📋 **Structured answers** — Every response follows a consistent legal format (jurisdiction, rights, deadlines, costs, next steps)
- 🔊 **Text-to-speech** — Listen to any answer aloud
- 📱 **Responsive** — Works on desktop and mobile
- ⚡ **100% client-side** — No backend, no data collection, your API key stays in your browser

## 🚀 Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/lexai.git
cd lexai
```

### 2. Open in browser

```bash
# No build step needed — just open index.html
open index.html
# or serve locally:
npx serve .
```

### 3. Add your API key

Click the **⚙️ gear icon** (top right) and enter your [Google Gemini API key](https://aistudio.google.com/app/apikey).

> Your key is stored only in your browser's `localStorage` and never sent to any server other than Google's API.
